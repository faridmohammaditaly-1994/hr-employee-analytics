# 👥 HR Employee Analytics

## Business Problem
Analyze employee attrition, compensation fairness, and performance
patterns to help HR reduce turnover and optimize talent management.

## Dataset
- 1,470 employees | 35 columns | PostgreSQL
- Source: IBM HR Analytics (Kaggle)

## Key Questions Answered
- Which departments and roles have the highest attrition?
- Is compensation fair across gender and job levels?
- What factors predict employee attrition?
- How does tenure relate to performance and satisfaction?
